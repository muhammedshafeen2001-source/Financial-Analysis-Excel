# Financial Analysis using Excel
This project contains a practical Income Statement financial analysis model built in Excel

## Contents
- Income Statement  (2-year comparison)
- EBITDA, EBIT, Net Profit calculation
- Key profitability ratios
- Analyst-style insights

  ## Tools Used
  Microsoft Excel

  ## Skills Demonstrated
  Financial Statement Analysis
  Ratio Analysis
  Excel Financial Modeling

  ## Data Source
  Publicly available financial data ( for learning purposes only)
